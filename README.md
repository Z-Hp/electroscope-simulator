# شبیه‌ساز الکتروسکوپ — راهنمای نصب

## پیش‌نیازها

ابتدا باید **Node.js** را روی کامپیوتر خود نصب کنید.

### نصب Node.js (اگر ندارید)

۱. به سایت https://nodejs.org بروید
۲. نسخه **LTS** (نسخه پایدار) را دانلود کنید
۳. فایل را اجرا و نصب کنید (Next → Next → Install)
۴. پس از نصب، کامپیوتر را ری‌استارت کنید

برای بررسی نصب، Command Prompt (cmd) یا PowerShell را باز کنید و تایپ کنید:
```
node --version
npm --version
```
اگر نسخه‌ها نمایش داده شدند، یعنی نصب موفق بوده.

---

## اجرای پروژه

### مرحله ۱: استخراج فایل
فایل zip دانلود شده را در یک پوشه استخراج کنید (مثلاً روی Desktop).

### مرحله ۲: باز کردن ترمینال در پوشه پروژه
- در پوشه استخراج شده، در نوار آدرس File Explorer تایپ کنید: `cmd` و Enter بزنید
- یا: کلید Shift را نگه دارید و راست‌کلیک کنید → «Open PowerShell window here»

### مرحله ۳: نصب وابستگی‌ها
در ترمینال تایپ کنید:
```
npm install
```
صبر کنید تا نصب کامل شود (ممکن است چند دقیقه طول بکشد).

### مرحله ۴: اجرای پروژه
```
npm run dev
```

### مرحله ۵: باز کردن در مرورگر
مرورگر را باز کنید و به آدرس زیر بروید:
```
http://localhost:3000
```

## مشکل‌یابی

### ارور «next is not recognized»
```
npx next dev
```

### ارور مربوط به پورت ۳۰۰۰ اشغال است
در فایل `package.json`، پورت ۳۰۰۰ را به ۳۰۰۱ تغییر دهید:
```
"dev": "next dev -p 3001"
```

### ارور مربوط به Prisma
```
npx prisma generate
```
